Bootstrap your infrastructure

Provision Dokku on DigitalOcean at example.com with Postgres and Redis by running the following commands: 

{{exports}}curl -sSL "{{url}}" | bash

And deploy your app:

git push {{platform.toLowerCase()}} master

How does it work?

Provision.sh leverages powerful open source tools like Terraform and Dokku or Flynn to build your own platform as a service on most cloud providers.

It helps you get started by generating and running for you the code of your infrastructure. And it can also teach you how to do it yourself if you read the script.
